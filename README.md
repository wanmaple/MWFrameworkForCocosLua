# MWFramework (LUA)

## Introduction

This is a cross-platform game framework for cocos2d-x 3.x lua.

## Support

- MVC scheme.
- ZIP compression and encryption.
- Sqlite3 database.
- Some common crypto methods.
- JSON format.
- Protobuf (use cloudwu/pbc).
- GIF texture.
- AStar path finder.
- Network scheme.
- Http service.
- SocketIO service.
